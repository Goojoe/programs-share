5.介绍:
Internet Download Manager（简称IDM）是一个用于Windows系统的下载管理器
IDM可以让用户自动下载某些类型的文件，可将文件划分为多个下载点以更快下载，并行出最近的下载，方便访问文件。相对于其他下载管理器，具有独特的动态文件分割技术。
